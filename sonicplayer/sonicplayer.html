SonicPlayer

Description:

Play music from games for the Sega Master System, Genesis and other systems!

SonicPlayer is a standalone music player for Windows supporting the VGM format. Files in this format can be created by many emulators, including Meka and Kega Fusion. Many VGM files are also available in archives, for example here and here.

SonicPlayer features a highly accurate implementation of the SN76489 PSG, thanks to this information. Also suported are the YM2413, YM2612 and YM2151 FM chips, using the implementations from the MAME project. SonicPlayer thus supports the playback of music written for many systems, including the Sega Genesis (Mega Drive), Master System and Game Gear, the BBC Micro and some arcade games.

System Requirements:

  • Microsoft .NET Framework 2.0
  • Up-to-date DirectX Runtime
